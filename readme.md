## Wake Word Detection System
MEMS microphone preamplifier with automatic gain control (AGC) and anti-aliasing filter for always-on voice wake-word detection in smart home devices.
The hardware interface is created with all analog devices and the software component is an ML Model running on an edge device, the Raspberry Pi 4.
Created as part of a project in the course Electronics Workshop 2 at IIIT Hyderabad.
