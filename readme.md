## Wake Word Detection System
MEMS Microphone Pre-Amplifier with Automatic Gain Control (AGC) and Anti-Aliasing Filter for always-on voice wake-word detection in smart home devices. \
The hardware interface is created with all analog devices and the software component is a speech detection ML Model running on an edge device, the Raspberry Pi 4, API calling Gemini Flash for queries. \
